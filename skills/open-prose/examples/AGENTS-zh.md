## 关键示例说明

### 01-hello-world.prose (推荐起点)
最简单的 OpenProse 程序，展示基础语法和会话执行。

### 28-gas-town.prose (多代理编排)
展示复杂多代理协作模式，Gas Town 是标志性示例。

### 29-captains-chair.prose (船长椅模式)
持久编排器模式，船长协调其他代理但不直接编写代码。

### 37-the-forge.prose (最复杂示例)
观看 AI 从零构建浏览器，展示 OpenProse 的强大能力。

### 44-run-endpoint-ux-test.prose (测试模式)
用户体验测试模式，包含 WebSocket 观察者和文件系统观察者，是测试模式的典范。

## 使用方法
```bash
# 运行特定示例
prose run examples/01-hello-world.prose

# 按关键词查找示例
# hello world → 01-hello-world.prose
# gas town → 28-gas-town.prose  
# captain → 29-captains-chair.prose
# forge → 37-the-forge.prose
# parallel → 16-parallel-reviews.prose
# error → 22-error-handling.prose
```

## 注意事项
- 示例兼具测试功能，运行示例可验证 OpenProse 功能
- UX 测试模式 (44) 展示了完整的测试基础设施模式
- 示例按数字顺序排列，建议按顺序学习
- 所有示例都遵循 OpenProse 最佳实践和约定
