# 路线图示例

这些示例展示了**计划中**但**尚未实现**的 OpenProse 语法。

它们被包含在内是为了展示语言的发展方向并收集设计反馈。

## 计划中的功能

| 功能 | 状态 | 示例文件 |
|---------|--------|--------------|
| 代理定义 | 计划中 | `simple-pipeline.prose` |
| 命名会话 | 计划中 | `simple-pipeline.prose` |
| 并行执行 | 计划中 | `parallel-review.prose` |
| 变量和上下文 | 计划中 | `iterative-refinement.prose` |
| 循环和条件语句 | 计划中 | `iterative-refinement.prose` |
| 导入 | 计划中 | `syntax/open-prose-syntax.prose` |

## 请不要运行这些示例

这些文件在当前解释器中无法运行。它们仅供参考。

有关可运行的示例，请查看父目录 `examples/`。
